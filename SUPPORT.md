# Support

This project is maintained as an open source documentation standard.

## Getting Help

Use GitHub issues for:

- questions about applying the standard to a repo
- confusing templates or rules
- validation failures
- platform adapter requests
- examples that do not match real agent workflows

Before opening an issue, check:

- [`README.md`](./README.md)
- [`skills/repo-memory/SKILL.md`](./skills/repo-memory/SKILL.md)
- [`skills/repo-memory/references/templates.md`](./skills/repo-memory/references/templates.md)
- [`skills/repo-memory/references/docs-structure-rules.md`](./skills/repo-memory/references/docs-structure-rules.md)
- [`skills/repo-memory/examples/`](./skills/repo-memory/examples/)

## Security Reports

Do not open public issues for security-sensitive problems. Follow
[`SECURITY.md`](./SECURITY.md) for reporting instructions.

## Scope

Maintainers can help with the Repo Memory standard itself. They cannot
guarantee support for every target repository, AI agent, or editor integration.
