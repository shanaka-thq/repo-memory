# Superpowers Bridge Example

This example shows how Repo Memory can complement Obra Superpowers.

The Superpowers spec and plan stay in `docs/superpowers/`. The Repo Memory
feature doc links those artifacts, records the accepted disposition, and keeps
the current implementation and handoff state in the feature doc.

## Files

- [`docs/features/search-refresh.md`](./docs/features/search-refresh.md) — canonical feature state and handoff surface.
- [`docs/superpowers/specs/2026-04-30-search-refresh.md`](./docs/superpowers/specs/2026-04-30-search-refresh.md) — companion spec artifact.
- [`docs/superpowers/plans/2026-04-30-search-refresh.md`](./docs/superpowers/plans/2026-04-30-search-refresh.md) — companion implementation plan artifact.
