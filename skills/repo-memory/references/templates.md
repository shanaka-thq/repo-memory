# Template Index

This directory contains split template files for Repo Memory documentation standards.

> **Canonical definitions:** Field names, allowed status values, and metadata schemas are defined in [STANDARD.md](../STANDARD.md) and [documentation-metadata-schema.md](./documentation-metadata-schema.md). These templates show example usage — defer to those files when definitions differ.

## Usage

Use these templates when a repository does not already define a stronger format. Follow the baseline structure and rules in `structure-and-setup.md` first, then apply specific templates as needed.

For full adoption guidance, see [SKILL.md](../SKILL.md).
