# Code of Conduct

## Our Standard

This project welcomes practical, respectful collaboration from people using
different editors, agents, platforms, and workflows.

Expected behavior:

- focus criticism on the work, not the person
- explain tradeoffs clearly
- assume good intent while still asking for evidence
- keep discussions relevant to the documentation standard
- avoid harassment, personal attacks, and discriminatory language

Unacceptable behavior includes:

- threats or harassment
- insults, slurs, or demeaning comments
- publishing private information without permission
- repeated off-topic disruption
- bad-faith contributions that weaken user safety or agent reliability

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other
contributions that violate this code of conduct. Serious or repeated violations
may result in a temporary or permanent ban from project spaces.

To report a concern, contact the maintainers through the repository's configured
GitHub reporting channels.
